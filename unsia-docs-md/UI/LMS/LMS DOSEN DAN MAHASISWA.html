Belum Dikerjakan Diverifikasi oleh Prodi & BPM Deadline: Sen, 8 Jun 2026 · 23:59

Instruksi Tugas

Buatlah sebuah halaman HTML statis sederhana yang memuat minimal 5 elemen semantik HTML5 dengan ketentuan sebagai berikut:

  1. Gunakan tag <header>, <nav>, <main>, <article>, dan <footer>
  2. Sertakan minimal 1 gambar dengan atribut alt yang deskriptif
  3. Tambahkan tabel sederhana dengan minimal 3 baris dan 3 kolom
  4. Pastikan kode tervalidasi di W3C Validator
  5. Tuliskan kode Anda langsung di lembar jawaban di bawah (atau lampirkan file .html jika perlu)

Bobot Nilai: 100 poin. Penilaian: Struktur HTML (40), Semantik (30), Estetika & Kreativitas (30).
